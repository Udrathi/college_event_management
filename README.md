# College Event Management System 🎓

A modern **College Event Management Web Application** built as a **PBL (Project-Based Learning)** project.  
The system allows **students and admins** to manage, register, and track college events with QR tickets, certificates, analytics, and calendar integration.

---

## 🚀 Features

### Authentication & Roles
- Student Signup / Login
- Admin Signup / Login
- Role-Based Access Control

### UI / UX
- Fully Responsive Design (Mobile + Desktop)
- Modern Navbar and Dashboard Layout
- Event Cards with Poster Images
- Light / Dark Mode Toggle

### Event Management (Admin)
- Create, Edit, Delete Events
- Event Fields:
  - Title
  - Date & Time
  - Venue
  - Description
  - Capacity
  - Category
  - Poster Image

### Event Registration (Student)
- Register / Unregister for Events
- Capacity Validation
- Waitlist Logic (Basic)

### QR Ticket System (Compulsory)
- Unique Ticket ID per Registration
- Auto-Generated QR Code
- Downloadable Ticket
- QR Visible in Student Dashboard

### Dashboards

**Admin Dashboard**
- Total Events
- Total Registrations
- Upcoming Events
- Participant Statistics

**Student Dashboard**
- Registered Events
- Ticket / QR Download
- Event History

### Certificates
- Auto-Generated PDF Certificate
- Includes Student Name, Event Name, Date
- Download Option After Event Completion

### Calendar Integration
- “Add to Calendar” Button
- Google Calendar Link / `.ics` File Support

### Analytics
- Participants per Event Chart
- Category Popularity Chart

### Mock Data
- Auto-Generated 10–15 Mock Users & Events on First Run
